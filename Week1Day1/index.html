HyperText Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

Objectives

Understand the theoretical components of the web and how they interact Understand the client-server model Understand the most important parts of what is displayed on the browser Understand the roles of HTML vs. CSS vs. Javascript Understand how to incrementally break down building a web page into its fundamental components
